Projet pour le ConHacks

Dans un monde ou l'internet des objets est de plus en plus présent, nous vous proposons ce mini-projet afin d'apprendre ou de connecter des objets intelligents qui sont chez vous

Le projet est divisé en plusieurs parties : 

- Arduino : On utilise une carte Arduino 101 ainsi que différents capteurs (intensité du son et température).

- Server : Il permet d'intéragir avec Alexa, notre amie qui se trouve enfermé dans l'amazon écho.


À défaut d'avoir utilise de l'ethernet, nous avons utilisé le serialport afin de communiquer entre l'arduino et le server.
